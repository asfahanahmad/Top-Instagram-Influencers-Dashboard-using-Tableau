# 📱 Top Instagram Influencers Dashboard using Tableau

## 📖 Project Overview

This interactive Tableau dashboard analyzes Instagram influencer performance using key social media metrics such as followers, engagement rate, likes, and audience distribution. The dashboard helps marketers, brands, and social media analysts identify top-performing influencers, compare engagement levels, and gain insights into influencer marketing trends.

---

## 📈 Key Performance Indicators (KPIs)

- Average Engagement Rate (60-Day)
- Total Followers
- Total Likes
- Average Likes per Post

---

## ✨ Dashboard Features

- Top 10 Influencers by Followers
- Top 10 Influencers by 60-Day Engagement Rate
- Top Countries by Influencer Count
- Followers vs Average Likes Scatter Plot
- Country Filter
- Influencer Channel Filter
- Interactive KPI Cards
- Dynamic Dashboard Filtering

## 📊 Business Insights

This dashboard helps users to:

- Identify the most-followed Instagram influencers.
- Compare influencers based on engagement rather than follower count.
- Analyze influencer distribution across countries.
- Evaluate the relationship between followers and average likes.
- Support influencer selection for marketing campaigns.
- Discover high-engagement creators with strong audience interaction.
